## 概要
このLINEボットは、〇〇（目的）を解決するために開発しました。具体的には、△△（ユーザーの問題やニーズ）に対して、リアルタイムで対応することを目指しています。このプロジェクトを通じて、API連携、ユーザーインタラクションの設計、さらにはボット開発に必要なスキルを習得しました。

製作中。
## 技術スタック
- **プログラミング言語**: Python, Node.js
- **LINE Messaging API**: ユーザーとのやり取りを実現するために利用
- **ホスティング/デプロイ**: AWS Lambda
## 開発で直面した課題と解決方法
- **ユーザー管理の課題**: ユーザーの状態を管理するために、LINEのプロフィール情報を元にセッション管理を行う必要がありました。Firebaseを使ってリアルタイムでデータを同期し、ユーザーごとの会話履歴を保持することができました。
- **非同期処理の実装**: ユーザーのリクエストに対して即時に応答するために、非同期処理を導入。バックエンドでの処理時間を短縮するために、AWS Lambdaを利用しました。
## ユーザー体験の向上
- ユーザーインターフェースの直感性を重視し、シンプルな対話式のインターフェースを設計しました。
- ユーザーがボットとの対話を続けたくなるように、リッチコンテンツ（画像、ボタンなど）を活用しました。
- ユーザーの過去の質問や回答を元にパーソナライズされた応答を提供する機能を実装しました。
